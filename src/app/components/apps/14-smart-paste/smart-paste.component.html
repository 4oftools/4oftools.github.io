简介

Smart Paste 是一款跨平台智能剪贴板应用，支持 Windows 与 macOS，自动记录剪贴历史、识别内容类型（文本、代码、链接等），并提供一键搜索与智能粘贴能力。

核心特性

支持 Windows 与 macOS，提供多设备一致的剪贴体验。自动分组文本、代码片段、图片与链接，支持固定常用片段与快捷搜索。

使用场景

适合程序员整理代码片段、产品/运营收集文案素材、日常多窗口复制粘贴等高频场景，让「复制一次、多处复用」变得更轻松。
